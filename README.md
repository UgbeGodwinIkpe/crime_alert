# Crime Alert System

A mobile and web-based crime reporting system developed using Flutter, Node.js, Express.js, and MongoDB.

## Features

- User Registration and Login
- Crime Reporting
- Crime Map Visualization
- User Profile Management
- Admin Dashboard
- Report Status Management
- User Management
- Analytics Dashboard

## Technologies

### Frontend
- Flutter
- Dart

### Backend
- Node.js
- Express.js
- MongoDB
